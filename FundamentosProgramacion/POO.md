# Lo aprendido del video: "Programación orientada a objetos (POO)"


## Pilares POO


### Lo que entendí:

Se entiende por:

Abstracción: Es el hecho de elegir y simplificar todos los atributos y métodos principales que definen una clase.

Encapsulamiento: Es la protección de atributos y métodos para evitar la modificación, volviendolos privados, dejando los demás en estado público.

Herencia: Es tener una clase como padre, donde contiene los atributos principales de sus clases hijas, es decir, comparte atributos y métodos con otras clases tal como se indica; una herencia. Esta permite generalizar para no repetir atributos entre clases.

Polimorfismo: Esto en la herencia, permite utilizar un método de distintas maneras por clases.
