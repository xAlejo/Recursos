# Operaciones Basicas para Python

## Tipos de datos

Los tipos de datos principales son:

* string (texto)
* int (números enteros)
* float (decimales)
* bool (verdadero o falso)

## Operaciones basicas de ejemplo

Suma:
print("suma:", 4 + 4)


Resta:
print("resta", 8 - 2)


Multiplicación:
print("multiplicacion", 4 * 3)


División:
print("division", 10 / 2)
