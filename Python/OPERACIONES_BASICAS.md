# Operaciones Basicas para Python

## Tipos de datos

Los tipos de datos principales son:

* string (texto)
* int (números enteros)
* float (decimales)
* bool (verdadero o falso)

## Operaciones basicas de matematicas

Suma:
print(5+3)


Resta:
print(8-2)


Multiplicación:
print(4*3)


División:
print(10/2)
