# Operaciones Basicas para Python

## Tipos de datos

Los tipos de datos principales son:

* string (texto)
* int (números enteros)
* float (decimales)
* bool (verdadero o falso)

num_1 = 5
num_2 = 3

* Suma
 print("suma:", num_1 + num_2)

* Resta
 print("resta:", num_1 - num_2)

* Multiplicacion
print("multiplicación:", num_1 * num_2)

* División
print("división:", num_1 / num_2)

* Modulo
print("módulo:", num_1 % num_2)

* Exponente
print("exponente:", num_1 ** num_2)

* Mayor que
print("mayor que",1 > 2)

* Menor que
print("menor que",1 < 2)

* Igual a
print("igual a",1 == 2)

* Diferente de
print("diferente a",1 != 2)

* Mayor o igual 
print("mayor o igual que",1 >= 2)
