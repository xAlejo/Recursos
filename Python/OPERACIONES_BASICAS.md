# Operaciones Basicas para Python

## Tipos de datos

Los tipos de datos principales son:

* string (texto)
* int (números enteros)
* float (decimales)
* bool (verdadero o falso)

```python
# Suma:
print("suma:", 4 + 4)

# Resta:
print("resta:", 8 - 2)

# Multiplicación:
print("multiplicación:", 4 * 3)

# División:
print("división:", 10 / 2)

# Módulo:
print("módulo:", 10 % 3)

# Exponente:
print("exponente:", 2 ** 3)

# Mayor que:
print("mayor que:", 5 > 3)

# Menor que:
print("menor que:", 2 < 4)

# Igual a:
print("igual a:", 4 == 4)

# Diferente de:
print("diferente de:", 4 != 2)
```
