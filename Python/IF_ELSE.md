# If Else en Python

## Definiciones
Las estructuras condicionales permiten tomar decisiones en Python según si una condición es verdadera o falsa.

* if se utiliza para ejecutar una instrucción cuando una condición es verdadera.
* elif se utiliza para evaluar una nueva condición si la condición anterior fue falsa.
* else se utiliza para ejecutar una instrucción si ninguna condición anterior se cumple.



## Operadores de comparación

- == igual a  
- != diferente de  
- > mayor que  
- < menor que  
- >= mayor o igual  
- <= menor o igual
```python
## Ejemplo
# If
edad = 18

if edad >= 18:
    print("Mayor de edad")


# If Else
edad = 16

if edad >= 18:
    print("Mayor de edad")
else:
    print("Menor de edad")


# Comparación mayor que
if 8 > 5:
    print("8 es mayor que 5")


# Comparación igualdad
if 4 == 4:
    print("Son iguales")
```
