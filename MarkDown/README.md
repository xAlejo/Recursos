# Tips Extras <br>

## Tips <br>
ejemplos útiles

### 📝 Estilos de texto

* `**texto**` → se usa para poner texto en **negritas**
* `*texto*` → se usa para poner texto en *cursiva*
* `***texto***` → se usa para poner texto en ***negrita y cursiva***
* `~~texto~~` → se usa para poner texto ~~tachado~~

---

### 📏 Espaciado Y Separador

* `<br>` → se usa para hacer saltos de línea o separar texto

Ejemplo:
`Hola 1 <br> Hola 2`

* `---` → se usa para crear una línea divisora (separar secciones)

Ejemplo:
`---`

---

### 🎥 Enlace a video de YouTube con imagen

* `[![Alt text](URL_IMAGEN)](URL_VIDEO)` → se usa para convertir una imagen en un enlace

Ejemplo:
`[![Alt text](https://img.youtube.com/vi/c-6V5OoD_nQ/0.jpg)](https://www.youtube.com/watch?v=c-6V5OoD_nQ)`

---

### 🖼️ Imágenes

* `![Texto alternativo](URL)` → se usa para insertar imágenes

Ejemplo:
`![Visual Studio Code](https://github.com/xAlejo/Recursos/blob/a221e9843b86d814c2c3d35fe42cb2f01bb14d0c/Visual_Studio_Code_1.35_icon.svg.png)`

---

### 💻 Bloques de código

* `` `texto` `` → se usa para crear cuadros grises (código en línea)
* ` ` → se usa para bloques de código más grandes

---

💡 Estos tips te ayudan a entender y reutilizar Markdown fácilmente en tus README.
