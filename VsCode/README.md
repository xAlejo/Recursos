#  **VISUAL STUDIO CODE** <br><br>
¿Que es visual studio code? <br>
Es un editor de codigo fuente que combina la siplicidad con la eficiencia el cual fue creado por Microsoft. <br>
¿Para que sirve? <br>
* Sirve para identificar y corregir errores <br>
Gestion de proyectos y carpetas de trabajo <br>
Seguimiento y control de versiones del codigo <br>
Trabajar en equipo con otros desarrolladores de manera remota y sincronizada <br>
Sirve para trabajar varios lenguajes de programacion y frameworks
Para desarrollar aplicaciones web y front-ed <br>
## **¿QUE LO HACE TAN ESPECIAL?** <br><br>
lo que lo hace especial es que es un programa de codigo abierto y gratuito, que contiene una experiencia de desarrollo rápida, fluida y altamente personalizable, tambien podemos destacar el soporte para múltiples lenguajes, su extensa biblioteca de extensiones, el sistema de depuración integrado y su compatibilidad con Git para un control de versiones eficiente. <br>
### **¿Y PORQUE ELEGIRLO?** <br>
porque es ideal tanto para principantes como para profesionales y sobre todo por sus funciones de autocompletado, resaltado de sintaxis y colaboración en tiempo real pueden ayudar a escribir un mejor código en menos tiempo.
