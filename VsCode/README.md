#  **VISUAL STUDIO CODE** <B>
¿Que es visual studio code? <b>
Es un editor de codigo fuente que combina la siplicidad con la eficiencia el cual fue creado por Microsoft. <b>
¿Para que sirve? <b>
Sirve para identificar y corregir errores <b>
Gestion de proyectos y carpetas de trabajo <b>
Seguimiento y control de versiones del codigo <b>
Trabajar en equipo con otros desarrolladores de manera remota y sincronizada <b>
Sirve para trabajar varios lenguajes de programacion y frameworks
Para desarrollar aplicaciones web y front-ed <b>
## **¿QUE LO HACE TAN ESPECIAL?** <b>
lo que lo hace especial es que es un programa de codigo abierto y gratuito, que contiene una experiencia de desarrollo rápida, fluida y altamente personalizable, tambien podemos destacar el soporte para múltiples lenguajes, su extensa biblioteca de extensiones, el sistema de depuración integrado y su compatibilidad con Git para un control de versiones eficiente. <b>
### **¿Y PORQUE ELEGIRLO?** <b>
porque es ideal tanto para principantes como para profesionales y sobre todo por sus funciones de autocompletado, resaltado de sintaxis y colaboración en tiempo real pueden ayudar a escribir un mejor código en menos tiempo.
